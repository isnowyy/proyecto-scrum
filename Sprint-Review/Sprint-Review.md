# Sprint Review

## Sprint
Sprint 1

## Fecha
13 de marzo de 2026

## Objetivo del Sprint
Desarrollar las funcionalidades principales de proyecto como crear tareas, listar, marcar como completadas, guardar tareas en json, cargar al iniciar las tareas, borrar tareas ya guardadas y archivo readme donde se explica el proyecto, problema, objetivo, tecnologias y instalacion.

---

## Funcionalidades completadas

Durante este sprint el equipo logró implementar las siguientes funcionalidades:

- Crear tareas
- Listar tareas
- marcar como completadas 
- Guardar tareas en archivo JSON
- cargar al iniciar las tareas
- borrar tareas 
- readme (informacion general del proyecto)


---

## Demostración del producto

Se realizó una demostración del sistema mostrando:

- La creación de una tarea desde la terminal
- La visualización de las tareas guardadas
- El almacenamiento de las tareas en un archivo JSON
- borrando  una tarea ya creada
- opcion de marcar como completada




---

## Feedback recibido

- Se sugirió agregar la opción de **marcar tareas como completadas**.
- También se recomendó mejorar la validación de los datos ingresados por el usuario.
- se sugirio incluir en el codigo el manejo de errores para que al momento de la ejecucion nunca se rompa el codigo
